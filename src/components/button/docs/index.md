# Button 按钮

## API

| 属性 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| type | 设置按钮类型 | string | `default` `primary` `danger` `ghost` | `default` |
| size | 设置按钮大小 | string | `small` | - |
| icon | 设置按钮的图标类型 | string | 见`Icon`组件 | - |
| disabled | 按钮失效状态 | boolean | `true` `false` | `false` |
| onClick | `click` 事件的 handler | function | - | - |
